<strong>Vector Slider</strong> - is a WordPress responsive jquery slider plugin. Vector Responsive Slider is a completely new and advanced way to represent your images in slides . Creation of image sliders with this wordpress plugin is fast and easy. Add your beautiful images into the Responsive Slider, after which give some nice styling and enjoy your modern slider. Responsive Slider helps you to display unlimited number of images to showcase your amazing works.

== Installation ==
1. Unzip vector-slider.zip
2. Upload all the files to the /wp-content/plugins/vector-slider.
3. Activate the plugin through the 'Plugins' menu in WordPress Admin.
4. For customizing the plugin's settings, click on Slider Settings under Slider Menu in Wordpress Admin Panel.
